# Data Structures and Algorithms Lab

This repository contains my implementations of various **Data Structures and Algorithms (DSA)** programs developed as part of my **second-year DSA laboratory coursework**.

I am currently pursuing a **Bachelor's degree in Electronics and Telecommunications Engineering**, and this repository serves as a collection of the algorithms and data structure concepts I implemented while learning DSA.

## Languages Used

* C++

## Topics Covered

Some of the topics implemented in this repository include:

* Recursion
* Arrays
* Searching Algorithms
* Sorting Algorithms
* Linked Lists
* Stacks and Queues
* Basic Algorithm Analysis

## Purpose

The objectives of this repository are to:

* Practice implementing fundamental data structures and algorithms
* Improve problem-solving and programming skills
* Maintain a record of laboratory assignments and experiments
* Build a structured reference for future learning
